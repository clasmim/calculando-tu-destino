# Calculando tu destino...
Statistical analysis of the quality of life of different countries in Europe in order to assess a recommended student exchange destination for the user based on different life factors.

## Objective:
Help the user make a decision on what country would be the best to choose for their student exchange. 

## Usage:
Simply use the sliders to select a value of the corresponding life factor and sumbit it. In the 'Results' tab will be the list of the top 3 countries recommended by our model.

## Tools and libraries used:
